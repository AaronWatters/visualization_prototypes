Embryo gene expression proof of concept

This page illustrates gene expression in a developing embryo over time.

Wg ⬤ ; Byn ⬤ ; Hkb ⬤ ; Tll ⬤ ; Fkh ⬤ ;

The GIF shown above is a fixed animation of the visualization. To run the visualization interactively please upload an appropriate data file below.

Load a JSON data file to initialize the visualization. The file must be preprocessed to group the gene expression information by timestamp. A script for creating the appropriate input file is here: https://github.com/AaronWatters/visualization_prototypes/blob/main/docs/embryo_0/make_genes_by_time_json.py

After the visualization is loaded, adjust the time slice using the slider or rotate the view by dragging in the canvas.
